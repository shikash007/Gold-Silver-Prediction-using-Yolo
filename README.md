# Gold-Silver-Prediction-using-Yolo
This Prediction Model is made up of Yolo model. 
It Forecast Gold and Silver price rate for 7 days.
The Price values is in dollars.
It will calculate 1 Ounce of Gold & Silver. 
1 Ounce = 28 gram
